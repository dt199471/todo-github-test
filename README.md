# シンプルなTODOリスト

GoogleのHPのようなシンプルでクリーンなデザインのTODOリストアプリケーションです。

## 機能

- ✅ TODOの追加・削除
- ✅ 完了/未完了の切り替え
- ✅ 期限の設定と表示（期限切れは赤色で表示）
- ✅ 画像の登録と表示
- ✅ ローカルストレージによるデータの永続化

## 使い方

1. `index.html`をブラウザで開く
2. TODOの内容を入力
3. 必要に応じて期限と画像を設定
4. 「追加」ボタンをクリック

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
